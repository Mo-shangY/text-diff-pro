# TextDiff Pro

A modern text comparison tool built with Vue 3, Vite, and Monaco Editor.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FMo-shangY%2Ftext-diff-pro)

## Features
- Side-by-side text comparison
- Syntax highlighting for multiple languages
- Dark/Light mode
- File upload support
- Responsive design

## Tech Stack
- Vue 3
- Vite
- Pinia
- Monaco Editor
- Tailwind CSS
